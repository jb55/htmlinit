
# htmlinit

Make a quick test page for your script:

`$ htmlinit <script.js>`

creates:

```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>Hello World</title>
    </head>
    <body>
        <h1>Hello World</h1>
    <script src="script.js"></script>
    </body>
</html>
```

## Install

    sudo make install

Will install under `/usr/local/bin`
